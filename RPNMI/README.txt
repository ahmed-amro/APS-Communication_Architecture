Content: 
- The RPNMI.py is the python script that implements the RPNMI algorithm. 
- The (*.xlsx) file contains the last output of the RPNMI.py tool. It reflects the results of a risk assessment activity for an Autonomous Passenger Ship.
- The remaing (*.csv) files represents the tables required for the RPNMI.py tool to run.
